### (1) video super-resolution 모델 연구개발
- 프로젝트 소개:
    - 초고해상도·초고품질 영상에 대한 video super-resolution 모델 개발.
- 프로젝트 진행:
    - PM 1명, 리서치 엔지니어 1명 (본인) 참여.
- 업무 내용:
    - 연구 및 실험:
        - latent vector 시각화하여 이미지의 해상도에 따른 원본 이미지와 latent vector 사이의 상관관계 확인.
        - latent vector normalization 유무에 따른 모델 학습 속도와 성능 비교.
        - latent vector의 국소 영역 및 특정 채널에 대한 조작이 재구축 이미지에 미치는 영향 분석.
        - 최신 Autoencoder 모델의 이미지 재구축 성능을 정량적으로 비교평가하여 최적의 모델 선정.
        - 'BasicVSR++' 모델을 기반으로 Autoencoder를 결합하여 latent space 도입, 계산 효율성 향상.
        - optical flow 후처리 방법이 video super-resolution 성능에 미치는 영향 비교.
    - 모델 학습 최적화:
        - 학습 데이터를 사전 전처리하여 파일 I/O 최소화 및 GPU 활용도와 학습 속도 최적화.
        - 8대의 RTX 4090과 `accelerate` 라이브러리를 활용하여 데이터 분산 학습 수행.
    - TIPS (Tech Incubator Program for Startup) 프로그램의 기술 부문 발표 자료 작성.

- 프로젝트 진행:
    - PO 1명, 백 엔드 개발자 2명, ML 엔지니어 1명 (본인) 참여.
- 주요 성과:
    - 이미지 임베딩 기반의 유사도 판단을 통해 중복 이미지의 77%를 자동으로 제거.
    - 수동 QC 전담 인력 고용 30% 절감.
- 업무 내용:
    - 이미지 회전 각도 분류 모델 개발:
        - 텍스트 이미지의 회전 각도를 0°, 90°, 180°, 270°로 분류하는 모델 개발.
        - 데이터: 자체 수집한 이미지 31,652장 및 EvArEST 데이터셋 510장 사용.
        - 모델 아키텍처: 사전 학습된 텍스트 탐지 모델을 분류 태스크에 맞게 일부 수정.
        - 테스트셋에 대한 정확도 88.2% 기록.
    - 아랍어 텍스트 이미지 QC 알고리즘 개발:
        - 높은 품질과 다양성을 가진 데이터를 수집하기 위해, 수동 QC 과정을 일부 자동화하는 알고리즘을 개발하여 비용 절감.
        - 회전된 이미지를 복원하는 분류 모델 사용.
        - 'Faiss' 라이브러리를 사용하여 이미지간 임베딩 기반 유사도 계산을 최적화하고 이를 사용해 기존에 수집된 이미지와 유사한 이미지를 자동으로 제외.
        - 'EasyOCR' 프레임워크를 활용하여 아랍어 텍스트의 수와 비율 확인.
        - RabbitMQ 기반 비동기 AMQP 서버 개발 및 백 오피스와 연동.
        - Slack과 연동하여 시스템 에러에 대한 실시간 알림 구현.

            - 발화 구간의 길이, 진폭 비율, 미녹음 주파수 수 등을 기준으로 음성 품질 평가.
            - waveform, spectrogram, mel-spectrogram으로 음성 데이터 시각화.

### (2) 아파트 실거래가 데이터 분석 (2020.01 ~ 2020.09)
- 프로젝트 소개:
    - 아파트 적정 분양가 산정 알고리즘 개발 가능성을 검토하기 위해, 내부 인력으로 데이터 분석을 시도하고 인사이트를 도출한 파일럿 프로젝트.
- 프로젝트 진행:
    - 데이터 분석의 전반적인 방향을 설정하고 팀을 이끌어 프로젝트를 주도.
    - 본인 외 전공 대학생 인턴 2~4명 참여.
- 주요 성과:
    - 동일한 조건에서 아파트 층에 따른 소비자의 적정 가격 인식을 수치화.
    - 부동산의 특성상 고전적인 방법론으로는 높은 정확도의 예측이 어려움을 확인.
    - 향후 예측 정확도를 위해서 타사에서 가지지 못한 당사만의 데이터 확보의 중요성 확인.
- 업무 내용:
    - web scraping을 통해 '네이버 부동산' 등에서 부동산 관련 데이터를 수집하고 데이터베이스 구축.
    - 공시지가, 교육환경 등 집값에 영향을 미치는 다양한 요인을 feature engineering 수행.
    - feature importance를 분석하여 핵심이 되는 feature를 선별.
    - 하나의 아파트 단지를 제외한 나머지 단지의 실거래가 데이터를 모델에 학습시킨 후 해당 단지의 실거래가를 예측하도록 하는 leave-one-out cross-validation 사용.
    - 예측 값이 ground truth보다 높을 때 더 큰 패널티를 주는 방식으로 loss function 설계해 모델 성능 향상.
    - 수집된 데이터를 'Mapbox'를 활용하여 시각화하여 직관적이 내용의 보고서 작성.
    - 최근 10년간 전용면적 및 층에 따른 소비자 선호도 변화를 분석.
