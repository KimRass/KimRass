      • 'Faiss' 라이브러리를 사용하여 이미지간 임베딩 기반 유사도 계산을 최적화하고 이를 사용해 기존에 수집된 이미지와 유사한 이미지를 자동으로 제외.
      • 'EasyOCR' 프레임워크를 활용하여 아랍어 텍스트의 수와 비율 확인.
      • RabbitMQ 기반 비동기 AMQP 서버 개발 및 백 오피스와 연동.
      • Slack과 연동하여 시스템 에러에 대한 실시간 알림 구현.

        • waveform, spectrogram, mel-spectrogram으로 음성 데이터 시각화.

(2) 아파트 실거래가 데이터 분석 (2020.01 ~ 2020.09)

  • 프로젝트 소개:
    • 아파트 적정 분양가 산정 알고리즘 개발 가능성을 검토하기 위해, 내부 인력으로 데이터 분석을 시도하고 인사이트를 도출한 파일럿 프로젝트.
  • 프로젝트 진행:
    • 데이터 분석의 전반적인 방향을 설정하고 팀을 이끌어 프로젝트를 주도.
    • 본인 외 전공 대학생 인턴 2~4명 참여.
  • 주요 성과:
    • 동일한 조건에서 아파트 층에 따른 소비자의 적정 가격 인식을 수치화.
    • 부동산의 특성상 고전적인 방법론으로는 높은 정확도의 예측이 어려움을 확인.
    • 향후 예측 정확도를 위해서 타사에서 가지지 못한 당사만의 데이터 확보의 중요성 확인.
  • 업무 내용:
    • web scraping을 통해 '네이버 부동산' 등에서 부동산 관련 데이터를 수집하고 데이터베이스 구축.
    • 공시지가, 교육환경 등 집값에 영향을 미치는 다양한 요인을 feature engineering 수행.
    • feature importance를 분석하여 핵심이 되는 feature를 선별.
    • 하나의 아파트 단지를 제외한 나머지 단지의 실거래가 데이터를 모델에 학습시킨 후 해당 단지의 실거래가를 예측하도록 하는 leave-one-out cross-validation 사용.
    • 예측 값이 ground truth보다 높을 때 더 큰 패널티를 주는 방식으로 loss function 설계해 모델 성능 향상.
    • 수집된 데이터를 'Mapbox'를 활용하여 시각화하여 직관적이 내용의 보고서 작성.
    • 최근 10년간 전용면적 및 층에 따른 소비자 선호도 변화를 분석.
