### (1) object detection 모델 개발
- 프로젝트 소개:
    - 경남 무인 도시철도 역사에 도입되어 유모차, 휠체어, 쓰러진 사람 등을 실시간으로 탐지하기 위한 object detection 모델 개발.
- 프로젝트 진행:
    - 본인 포함 ML 엔지니어 2명 참여.
    - 데이터 수집 및 처리 부문 담당.
- 주요 성과:
    - 영상 데이터를 지속적으로 활용하여 모델 성능을 개선할 수 있는 파이프라인 구축.
- 업무 내용:
    - 유료 데이터 및 'AI-Hub' [1]의 공공 데이터를 조사하여 유용한 데이터셋 선정.
    - 선정된 데이터셋을 모델 학습에 사용할 수 있도록 재현 가능한 데이터 전처리 환경 구축:
        - 멀티프로세싱을 도입하여 대용량 파일의 압축 해제 속도 최적화.
    - 대량의 실제 지하철 CCTV 영상으로부터 탐지 대상 객체가 포함된 장면을 효율적으로 추출하는 환경 구축:
        - 사전 학습된 모델을 사용해 일부 장면을 자동으로 선별하고 이를 사람이 검수할 수 있는 시스템 설계 및 구현.
        - 'PyQt' 라이브러리를 활용한 직관적인 GUI를 개발하여 사용자가 검수를 간편하게 수행할 수 있도록 지원.
